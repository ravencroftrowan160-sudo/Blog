# My Blog Website

A modern, responsive blog website built with HTML5, CSS3, and JavaScript.

## Project Structure

```
Blog/
├── index.html          # Home page
├── blog.html           # Blog listing page
├── about.html          # About page
├── contact.html        # Contact page
├── styles.css          # Main stylesheet
├── script.js           # JavaScript functionality
├── favicon.ico         # Website icon
├── images/             # Image assets folder
├── css/                # Additional CSS files (optional)
├── js/                 # Additional JavaScript files (optional)
└── README.md           # This file
```

## Features

- **Responsive Design**: Works beautifully on desktop, tablet, and mobile devices
- **Modern Layout**: Clean and professional design with smooth interactions
- **Navigation**: Easy-to-use navigation menu across all pages
- **Blog Section**: Dedicated page for blog articles with sidebar widgets
- **About Page**: Personal introduction and information about the author
- **Contact Form**: Contact page with form and social media links
- **Fast Loading**: Optimized CSS and JavaScript for performance

## Pages

1. **Home (index.html)** - Landing page with featured articles
2. **Blog (blog.html)** - Blog article listing with categories and subscription
3. **About (about.html)** - About the author, skills, and mission
4. **Contact (contact.html)** - Contact form and social media links

## Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Styling with Grid, Flexbox, and custom properties
- **JavaScript (ES6+)** - Interactive features and functionality

## Getting Started

1. Clone or download the repository
2. Open `index.html` in your web browser
3. Navigate through the pages using the navigation menu
4. Customize the content with your own information

## Customization

### Colors
Edit the CSS custom properties in `styles.css`:
```css
:root {
    --primary-color: #3498db;
    --secondary-color: #2c3e50;
    --text-color: #333;
    --light-bg: #f8f9fa;
}
```

### Content
Update the HTML files with your own:
- Blog articles and categories
- Personal information
- Contact details and social media links
- Images in the `images/` folder

### Typography
Modify font families and sizes in `styles.css` to match your preferences.
